# Certified two-shift witnesses

This directory records rigorous interval-arithmetic certificates produced by
`.github/workflows/two-shift-certificate.yml` and
`scripts/certify_two_shift_rank.py`.

For each `N` from 4 through 8, the corresponding JSON file contains explicit
shifts `s` and `t`, the selected maximal minor in each parity sector, and an
interval enclosure for its determinant. In every recorded case, both determinant
intervals exclude zero.

Therefore, for each fixed `N` in `{4,5,6,7,8}`, the simultaneous commutant of the
two parity-restricted Gaussian translations is scalar in each parity sector.
Equivalently,

```text
Comm(T_s, T_t) = span{I, J},
```

and the unital algebra generated by the two translations is

```text
M_{N_+}(C) direct-sum M_{N_-}(C).
```

Because the certified minors are real-analytic functions of `(s,t)` and are
nonzero at the recorded witnesses, maximal rank holds for generic `(s,t)` for
each of these five fixed dimensions. Here generic means outside a proper
real-analytic determinantal set.

## Scope

These files rigorously certify the statement only for `N = 4,...,8`. They do not
by themselves prove a dimension-uniform theorem for all `N`.
